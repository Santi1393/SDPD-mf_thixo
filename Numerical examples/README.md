In this folder you will find a collection of all the numerical cases covered in this research. Detailed information on each case and how to run them can be found in the README file in each example folder. The complete list of cases studied is detailed below:

Surface tension of a droplet - Used for static validation - See Figure 1_(a)
Retraction of a stretched droplet - Used for static validation - See Figure 1_(b)
Static contact angle between droplet and solid wall - Used for static validation - See Figure 1_(c)
Triple contact angle between three diferents droplets - Used for static validation - See detail in Figure 1_(c)
Poiseuille flow for one phase - Used for dynamic validation - See Figure A.12_(a) and A.12_(d)
Reverse Poiseuille flow for one phase - Used for dynamic validation - See Figure A.12_(b) and A.12_(e)
Flow around a cylinder - Used for dynamic validation - See Figure A.12_(c) and A.12_(f)
Poiseuille flow for two phases - Used for dynamic validation - See Figure 2_(a)
Taylor deformation vs capillary number - Used for dynamic validation - See Figure 2_(b)
Droplet break-up - Used for dynamic validation - See Figure 2_(c)
Thixotropic shear flow in a channel - Used for thixotropic validation - See Figure 3_(a) and 3_(b)
Liquid-Liquid Phase Separation (LLPS) - First exploratory case - See Figure 4 and 5
Poiseuille flow for two phases with one thixotropic phase - Second exploratory case - See Figure 6
Emulsion with thixotropic continuous phase and newtonian droplets - Second exploratory case - See Figure 7
Emulsion with newtonian continuous phase and thixotropic droplets - Second exploratory case - See Figure 8
Droplet dynamics in a periodically constricted channel - Third exploratory case - See Figure 9 and 10
